# Roadmap

## 1. Núcleo — Implementado

Funcionalidades actuales del sistema:

- [x] Autenticación y control de acceso por niveles jerárquicos (1–4).
- [x] Registro y gestión de asistencia de niños.
- [x] Registro y gestión de asistencia del personal.
- [x] Planificación de eventos.
- [x] Reportes de niños.
- [x] Comunicación con padres (llamada o mensaje de texto).
- [x] Control de cumpleañeros del mes (niños y personal).
- [x] Medidas disciplinarias para el personal (suspensiones, tiempos de espera).
- [x] Sistema de solicitudes para ingreso de nuevo personal (aprobación/rechazo por coordinador general).
- [x] Control de transiciones de niños entre grupos.
- [x] Control de graduaciones de niños.
- [x] Gestión de fichas (creación, notas, estado).
- [x] Administración de roles y requisitos.
- [x] Reglas de visibilidad por nivel jerárquico.
- [x] Asistencia cruzada entre turnos para maestros.
- [x] Badges de cumplimiento de requisitos para el personal.
